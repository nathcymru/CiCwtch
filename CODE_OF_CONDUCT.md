# Code of Conduct

## Purpose
CiCwtch aims to foster a professional, respectful, and structured environment.

## Expected Behaviour
- Be respectful and constructive.
- Focus on technical merit and architectural integrity.
- Avoid personal criticism.

## Unacceptable Behaviour
- Harassment or discrimination.
- Personal attacks.
- Disruptive or abusive language.

## Reporting
Report concerns confidentially to the project maintainer via email or private GitHub communication.
