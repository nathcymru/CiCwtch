Dogs belong to clients through `client_id`.

## Endpoints

- `GET /api/v1/dogs`
- `GET /api/v1/dogs/:id`
- `POST /api/v1/dogs`
- `PUT /api/v1/dogs/:id`
- `DELETE /api/v1/dogs/:id`

## Validation

- `client_id` is required and must reference an active client.
- `name` is required.
- `sex` must be one of `male`, `female`, or `unknown` when supplied.
