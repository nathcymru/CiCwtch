# CiCwtch - Invoice Lines API
## Line-item endpoints supporting invoice headers

Invoice lines are currently backend-only in Phase 1.

## Endpoints

- `GET /api/v1/invoice-lines`
- `GET /api/v1/invoice-lines/:id`
- `POST /api/v1/invoice-lines`
- `PUT /api/v1/invoice-lines/:id`
- `DELETE /api/v1/invoice-lines/:id`

## Validation

- `invoice_header_id` is required.
- `description` is required.
- `walk_id` is optional but validated when supplied.
- Invoice lines do not currently use soft deletion.
